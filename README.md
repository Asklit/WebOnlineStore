# Online Store

  ![Python](https://img.shields.io/badge/Python-3.8+-blue.svg)
  ![Flask](https://img.shields.io/badge/Flask-2.0+-green.svg)

  A web-based online store built using the **Flask** framework, designed to provide a seamless shopping experience with user authentication, product catalog, and cart functionality.

## Project Overview

  **Author**: Alexander Khanzhin  
  **Objective**: Create a fully functional online store website using Flask, enabling users to browse products, manage their cart, and complete purchases.

## Getting Started

### Prerequisites
  - Python 3.8 or higher

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/online-store.git
   cd online-store
   ```
2. Install dependencies:

   ```bash
   pip install -r requirements.txt
   ```

3. Run the application

   ```bash
   python main.py
   ```

4. Open your browser and navigate to:

   ```bash
   http://127.0.0.1:8080/
   ```

## Features

  | Feature                | Description                                                                 |
  |------------------------|-----------------------------------------------------------------------------|
  | **Main Menu**          | Access the store's homepage with a clean and intuitive interface.            |
  | **User Authentication**| Register or log in via dedicated pages in the top menu.                     |
  | **Product Catalog**    | Browse products and add items to the cart (requires login).                  |
  | **Shopping Cart**      | View, remove, or purchase items in the cart with a streamlined checkout process. |

## Technologies Used

  - **Backend**: Flask
  - **Frontend**: HTML, CSS (assumed, customizable)
  - **Dependencies**: Listed in `requirements.txt`

## Usage

  1. **Access the Store**: Visit `http://127.0.0.1:8080/` to land on the main menu.
  2. **Register/Login**: Use the top menu to create an account or sign in.
  3. **Browse Catalog**: Explore products and add desired items to your cart (login required).
  4. **Manage Cart**: In the cart section, remove items or proceed to checkout to complete your purchase.
